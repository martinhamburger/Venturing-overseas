# 下一步

## 第一优先级

1. 确认一个具体的 Figma 科研海报母版
2. 按母版结构检查 `poster-agent/templates/poster_sections.json`
3. 若模板结构不一致，先改模板映射，再改视觉原型

## 第二优先级

1. 补充最终署名
2. 补充学校或项目标识
3. 决定最终画幅：
   - A1 竖版
   - A0 竖版
   - 屏幕展示版

## 第三优先级

1. 将一版图表或示意图补进 `assets/figures/`
2. 在 `poster-agent/outputs/poster_handoff.md` 中补具体图片尺寸和位置
3. 做一次正式 Figma 导入并记录结果

## 升级路线

### V1

- 以模板填充为主
- MCP 主要负责读取和导入

### V2

- 增加对已有 Figma 模板节点的读取回写
- 让 Codex 根据截图和节点结构给出更细的修改建议

### V3

- 若后续确认桥接方案可靠，再尝试更强的自动写入能力
