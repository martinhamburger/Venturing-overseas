# 模板备注

## 当前状态

当前项目还没有绑定具体的 Figma 科研海报母版链接，因此模板层先采用“抽象模板 + 结构规则”的方式。

## 选模板时的优先标准

1. A1 或 A0 竖版科研海报
2. 支持明显的主标题区、结论区、结果区
3. 有足够空间放数字卡或对比图
4. 不依赖过重的装饰性背景

## 不建议的模板

- 区块太多、文字格子太碎的模板
- 把背景、文献、方法写成同等权重的模板
- 过于偏商业宣传海报的模板

## 当前兼容策略

- `poster/index.html` 适合作为“先导入再微调”的视觉原型
- `poster-agent/templates/poster_sections.json` 适合作为“先贴到已有模板”的结构清单
