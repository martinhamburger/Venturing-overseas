# 海报设计硬规则

## 1. 叙事规则

- 一张海报只允许一个主叙事：
  - 中国光伏产业仍具优势，但竞争已转向本地化、合规和全球经营能力
- 背景不是主角，结论和证据才是主角
- 读者应在 8 秒内知道“这张海报想证明什么”
- 结果区和结论区优先于背景铺垫，这一点优先级最高

## 2. 文案规则

- 标题尽量控制在 24 字以内；如果必须更长，拆成主标题和副标题
- 标题最好控制在两行内
- 核心结论区不超过 40 到 60 字
- 单个正文区块尽量不超过 90 到 120 字
- 优先使用项目符号，不优先使用长段落
- 一张海报中不要重复同一个论点超过两次
- 没有摘要必要时，不要强行保留大段摘要
- 除摘要外，尽量不要出现超过 3 句的连续文字块

## 3. 层级规则

- 结论区和关键数字必须在视觉上最先被看见
- 结果区优先级高于背景区
- 案例区和区域策略区属于支撑证据，应紧跟主结论
- 参考文献和联系信息必须弱化，不抢主视觉

## 4. 视觉规则

- 主色不超过 3 个
- 正文区域至少保持 24pt 的等效可读尺寸
- 版面优先采用 3 栏或 4 栏的清晰阅读路径
- 预留充分空白，目标是让海报至少约 1/3 区域保持留白感
- 不要让每个区块都一样重
- 必须保留足够留白
- 图表和数字卡优先于大段文字
- 如果出现表格，必须极度压缩，只保留最关键字段
- 正文字体数量尽量控制在 2 套以内
- 正文应保持高对比度，不使用浅色正文和低对比配色

## 5. Figma 执行规则

- 自动化阶段优先处理内容填充和结构对齐
- 最终字体、字距、对齐、裁切、阴影和导出边距必须人工检查
- 没有确认母版之前，不要重做整套版式系统
- 中文字体优先选用 Figma 可稳定调用的字体，如 `Noto Sans SC`、`Noto Serif SC`、`Microsoft YaHei`
