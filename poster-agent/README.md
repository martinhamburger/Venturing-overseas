# poster-agent

本目录是这次海报项目的中间工作层，目标不是直接替代 Figma，而是给 Codex 一个稳定、可复用、可审查的工作环境。

## 目录说明

- `codex_task.md`
  - 任务边界和输出要求
- `prompts/`
  - 给 Codex 的提示模板
- `rules/`
  - 版式、层级、Figma 映射规则
- `templates/`
  - 海报区块结构和版式预设
- `outputs/`
  - 当前课题的内容产物和交接说明
- `assets/`
  - 图表、校徽、品牌图形等素材

## 当前使用原则

1. 先写 `outputs/`
2. 再改 `poster/`
3. 最后导入 Figma

## 当前课题

`贸易战2.0背景下中国企业出海的应对策略研究`
